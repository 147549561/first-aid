# first-aid

## 说明
急救教程小程序，为用户提供心脏骤停和常见意外的应急教程，以及提供简单的求助功能。

## 项目结构
* assets 静态资源
* components 公共组件
* doc 文档
* libs 依赖
* pages — 存放项目页面相关文件
* utils — 存放utils文件，可require引入
